The Object

the loop · the tether · the mill

The Loop
5 IN → 1 PRIM → 1 OUT

The shape of the thing: five inputs are pressed through one primitive (the 'prim') to one output, deterministically — same input, same prim, same out, no variance. Dressed in the chrome of valence shells and stable orbits, but underneath it is a fold: many into one into one.

The Tether
every prim is signed

The working core is small and real: a local control plane (a tiny Python server) catches each event from the browser and commits it to a git repo — so every prim is logged, timestamped, and signed. The lattice 'responds to pressure, not requests'; the proof is the commit history.

The Mill
Ada's Mill, est. 1843

The manifesto: computation as deterministic milling, not probabilistic guessing — the page dates itself to 1843, the year of Lovelace's Notes, and signs off 'No AI. No GPU. No debate.' It is the F/M/B and 0root determinism creed, miniaturized into one obsessive object.
